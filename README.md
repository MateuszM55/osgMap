Map application to create students project with OpenSceneGraph environment

Build instructions:

1. Follow build instructions for OpenSceneGraph. When finished, set OSG\_DIR env variable if possible
2. git pull this repo
3. mkdir build
4. cd build
5. cmake ..
6. or  cmake -DOSG\_ROOT\_DIR=C:/path/to/OpenSceneGraph ..
7. Run building process

Run:

Set working directory with -path option set to directory with data to render eg. -path ./map\_data
